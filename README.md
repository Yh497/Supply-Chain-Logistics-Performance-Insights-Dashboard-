# Logistics Performance Insights Dashboard 🚛

An interactive Power BI dashboard designed to streamline supply chain operations, track shipment efficiency, and monitor logistics KPIs for data-driven decision-making.

## 🚀 Project Overview
This project transforms complex logistics and supply chain data into a visual command center. It provides visibility into delivery timelines, carrier performance, and distribution costs, helping businesses identify bottlenecks and optimize their fulfillment processes.

## 🛠️ Tech Stack
- **Power BI Desktop**: For end-to-end data modeling and interactive report design.
- **Power Query**: Used for data transformation, merging disparate shipping logs, and handling delivery timestamps.
- **DAX (Data Analysis Expressions)**: To calculate critical metrics like On-Time Delivery (OTD) rate, Average Lead Time, and Shipping Cost per Unit.

## 📊 Key Features & Analysis
- **Delivery Performance**: Tracking successful deliveries versus delays to improve customer satisfaction.
- **Fleet & Carrier Analysis**: Evaluating individual carrier efficiency and volume distribution.
- **Route & Geographic Insights**: Mapping shipping lanes to identify high-traffic regions and potential logistics hubs.
- **Inventory & Order Status**: Real-time monitoring of pending, in-transit, and delivered shipments.
- **Cost Optimization**: Analysis of freight expenses across different modes of transport (Air, Sea, Road).

## 📈 Dashboard Highlights
- **Dynamic Slicers**: Filter by Date Range, Warehouse Location, Carrier, and Shipment Priority.
- **KPI Cards**: Instant visibility into Total Shipments, Average Shipping Days, and Total Logistics Cost.
- **Trend Analysis**: Visualizing seasonal spikes in shipping volume to manage resource allocation.

## 📂 Repository Structure
- `Logistics Dashboard.pbix`: The core Power BI dashboard file.
- `/data`: Folder containing the logistics datasets (CSV/Excel).
- `/screenshots`: Visual previews of the dashboard’s reporting pages.
